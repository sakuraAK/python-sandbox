### Installing

* Open terminal or command prompt
* Navigate to your dev folder 
```
$ cd users/andre/dev
$ git clone https://github.com/sakuraAK/python-sandbox.git
$ cd python-sandbox
$ python -m  pip install --upgrade pip
$ pip install -r requirements.txt
```
